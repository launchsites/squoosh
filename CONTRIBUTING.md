# Contributing

Thanks for contributing!

## Development setup

```bash
npm install
npm run build
```

## Running the CLI locally

```bash
npm run dev -- ./images
```

## Global testing

```bash
npm link
squoosh ./images
```

## Coding style

- Keep output paths and naming rules consistent with README.
- Avoid adding new dependencies unless necessary.
- Favor readable, well-structured TypeScript.

## Reporting issues

Please include:

- macOS version
- Node.js version
- CLI command you ran
- Full error output
- Example input file and format selection
